### 78. Subsets (子集)

题目:
<https://leetcode.com/problems/subsets/>


难度:   Medium


**回溯法**


```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        """
        :type nums: List[int]
        :rtype: List[List[int]]
        """
        def backtrack(idx, curr=[]):
            output.append(curr)
            for i in range(idx, n):
                backtrack(i+1, curr+[nums[i]])
        
        output = []
        n = len(nums)

        backtrack(0, [])
        return output
```
